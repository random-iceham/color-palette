# Color Palette Generator

A **React app** that generates color palettes with different styles. Click on a color to copy its hex code!

---

### Live Demo
[https://random-iceham.github.io/color-palette/](https://random-iceham.github.io/color-palette/)

---

### Features
- Generate **random color palettes**.
- Choose from different **palette modes**:
  - **Random** – completely random colors.
  - **Distinct (Golden Ratio)** – evenly spaced colors using the golden ratio.
  - **Pastel** – soft, muted colors.
  - **Monochrome** – shades of a single color.
- **Copy hex color codes** with a click.
- **Responsive UI** built with **Tailwind CSS**.

- JavaScript (ES6+)
- Tailwind CSS
